# MR2 Robot 
## 湖北工业大学 ROBOCON 2019 Ylt

这是一个并联式的四足机器人 每条腿使用了两个电机驱动共四条腿，八个电机
期望实现四种步态，从慢到快依次为，行进步态（三足支撑）、小跑步态（两
拍步态）、溜蹄步态、疾驰步态，具有自动适应，通过陀螺仪实现自我平衡功
能，使用高精度陀螺仪积分计算距离实现定位##
